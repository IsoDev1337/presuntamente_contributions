# Roadmap interno

Esta carpeta descarga el `ROADMAP.md` raíz para que siga siendo operativo.

## Convención

- [`../../ROADMAP.md`](../../ROADMAP.md) es el documento vivo que se lee al empezar una sesión.
- [`historial-2026-05.md`](historial-2026-05.md) recoge cierres de sesión y trayectos largos de mayo de 2026.
- [`fases-cerradas.md`](fases-cerradas.md) resume fases, lotes y bloques ya completados.
- [`aprendizajes.md`](aprendizajes.md) conserva aprendizajes largos que no necesitan estar siempre en el roadmap principal.
- [`roadmap-completo-antes-division-2026-05-25.md`](roadmap-completo-antes-division-2026-05-25.md) conserva una copia íntegra del `ROADMAP.md` anterior a la división para no perder detalle histórico.

## Regla práctica

Si una actualización ayuda a decidir qué hacer ahora, va en `ROADMAP.md`.

Si explica cómo se llegó hasta aquí, va en el histórico.

Si se convierte en regla obligatoria, no va aquí: se promueve a `AGENTS.md`, `DESIGN.md` o `docs/diseno/`.
